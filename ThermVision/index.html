Introduction

This work contributes to the field of image synthesis by introducing a novel diffusion-based framework, for generating high-quality synthetic thermal face images using the Flux diffusion model. By integrating a video retargeting pipeline, we address challenges in synthesizing diverse facial expressions, head poses, and subject-specific thermal characteristics. A key outcome is the creation of an open-source synthetic thermal face dataset, optimized for AI-driven applications such as thermal gender classification, facial detection, and landmark detection. Our approach enhances data availability in thermal imaging, reducing reliance on costly real-world data collection.

The work is submitted in IEEE Conference.

This dataset includes face data localization annotations to support tasks such as emotion recognition, gender classfication and pose estimation. We have provided precise bounding box annotations for each face in the images, as well as key face annotation labels, to enable researchers and developers to train and evaluate advanced computer vision models. These annotations are essential for tasks requiring accurate face localization and are designed to be easy to integrate into machine learning workflows.

54 Keypoints Facial Landmark Detection

We have rigorously evaluated the rendered data by running facial landmark detection tests using a pretrained thermal landmark detector. This evaluation ensures that the dataset is suitable for thermal-based facial landmark detection tasks. The pretrained detector has been applied to our dataset, and the results have been analyzed for accuracy, robustness, and consistency across different poses and thermal conditions. This evaluation provides confidence in the quality and reliability of the dataset for use in thermal vision applications.
